# TO DO LIST CLI
## Todo-CLI

A minimalist, file-based command-line todo manager designed for developers who want to manage tasks without leaving the terminal.

## Install

Go to [https://github.com/travletothefurureprogramming/To-Do-CLI/releases/tag/Release](https://github.com/travletothefurureprogramming/To-Do-CLI/releases/tag/Release) and download the .exe

## How to run

Open the .exe and you are ready to go!!!
